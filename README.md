# SafeRoute SA 🛡️

A community-powered real-time safety map and neighbourhood 
watch app built for South Africa.

## What It Does
South Africans can report and track safety incidents in 
real time on a live map — crimes, accidents, suspicious 
activity — visible instantly to their community.

## Key Features
- Real-time incident reporting with geo-location
- Live safety map powered by Google Maps
- AI-powered incident categorisation (Azure AI)
- SMS alerts to nearby users via Twilio
- Community feed with live updates
- User authentication and profiles

## Tech Stack
- React 18, Tailwind CSS, Vite
- Firebase (Firestore, Auth, Storage)
- Google Maps API
- Azure OpenAI
- Twilio SMS
- Node.js backend with Express
- PostgreSQL + Prisma
